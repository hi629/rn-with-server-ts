以下の技術を使った TODO サンプルアプリです。

frontend: React Native
backend: nestjs, fastify, TypeORM

## Installation

### backend

```bash
$ cd backend
$ npm install
```

### frontend

```
$ cd rn-front
$ npm install
```

## Running the app

### backend

```bash
$ cd backend
$ npm run start

```

### frontend

```
$ cd rn-front
$ yarn start
```
