以下の技術を使った TODO サンプルアプリです。

frontend: React Native

backend: nestjs, fastify, TypeORM, sqlite

![rn-front](https://user-images.githubusercontent.com/28013909/125194763-9996e000-e28d-11eb-80f3-325393ffa551.png)


## Installation

### backend

```bash
$ cd backend
$ npm install
```

### frontend

```
$ cd rn-front
$ npm install
```

## Running the app

### backend

```bash
$ cd backend
$ npm run start

```

### frontend

```
$ cd rn-front
$ yarn start
```
